התוסף Elementor for Wizshop הינו הרחבה [לתוסף החנות](https://wizshop.co.il/plugin/) של חשבשבת.
כעת ניתן לעצב בקלות ופשטות תבניות לעמוד המוצר באלמנטור!


== Description ==

= הדרך הקלה והמקצועית ביותר להקים אתר מסחרי למשתמשי חשבשבת =
התוסף משלים באופן יוצא מן הכלל את חווית הקמת האתר המסחרי עם תוסף החנות של חשבשבת.
אתר מסחרי העובד עם תוסף חשבשבת לא מצריך הקמה של דאטה לחנות. כל ההקמה והניהול של המוצרים, הקטגוריות, הלקוחות, המלאי וכו', מתנהלים בתוכנת חשבשבת של בית העסק והתוסף "שואב" את המידע ומציג אותו באתר.
כמו כן הזמנות ורכישות מגיעות ישירות לחשבשבת ללא צורך בממשקים שונים. חשבונית מס מופקת בתוכנה  אוטומטית בעת רכישה באתר, ומסמך הזמנה מוקם בעת הזמנה.

עם התוסף שלנו לאלמנטור כל מה שנותר לכם לעשות, הוא לעצב ולבנות באלמנטור תבנית עיצובית ייחודית ומהממת לדפי החנות!
ללא שום מגבלות, תוכלו לעצב חווית UI/UX מקסימלית וייחודית לגולשים באתר שלכם!

= דפי מוצר ייחודיים כמו שרק אתם יודעים לעצב =
כעת אין פשוט יותר מלממש את החזון העיצובי שלכם לאתר מסחרי ללקוח.
בעריכת התבנית של דף המוצר, לכל פיסת מידע שמוזן בחשבשבת למוצר קיים רכיב מקביל.
כך תוכלו לגרור ולעצב בכל חלקי הדף את שם המוצר, המק"ט, סטטוס המלאי, כפתור ההוספה לסל , ועוד רכיבים רבים ומגוונים.
כל המידע על המוצר בידיכם העיצוביות בגרירת עכבר פשוטה!

= הזנת מידע ללא הגבלה לכל מוצר =
כמות, איכות ועיצוב המידע על המוצר אינם מוגבלים לטקסט המוזן בכרטיס הפריט בחשבשבת.
זהו אחד מיתורונותיו הגדולים של התוסף.
בעזרת רכיב תיאור ייחודי תוכל להציג לכל מוצר מידע נוסף, ללא הגבלה של תווים או רכיבים,  ולעצבו כראות עיניכם באלמנטור.
כל זאת ללא פגיעה במהות התוסף של חשבשבת המשכפל את דפי המוצר ומאפשר הקמת חנות ללא שימוש בדפים ממשיים היושבים בוורדפרס, מה שמקל רבות על משקל והקמת האתר.

= עיצובים שונים למוצרים שונים באמצעות תנאים =
אין מגבלה למספר התבניות שאפשר ליצור לדפי המוצר. תוכלו ליצור תבניות עיצוב שונות ולהגדירן לדפי מוצר ספציפיים.
תוכלו להחיל את התנאים לפי קטגוריה או לפי שמות המוצרים.
אין סוף אפשרויות לעיצוב מוצרים בחנות אחת!

= רכיבים גלובליים של החנות לשילוב בדפי האתר השונים =
בנוסף לרכיבי דף המוצר, תקבלו בתוסף רכיבים עצמאיים של החנות, אותם תוכלו לשלב בחלקי האתר השונים.
בראשם, כפתור העברה לסל, המזוהה לפי מפתח המוצר וישתלב בכל מקום באתר שתרצו.
דף ראשי, דפי נחיתה, דפי תוכן, ועוד...בכל אלו תוכלו לשלב הוספה לסל של מוצר לרכישה, אשר מקושר למערכת הרכישה באתר (סל הקניות, עמוד הרכישה)  ומשם למערכת הסליקה של חשבשבת.

= שילוב רכיבי חנות בקלות בהדר האתר =
אנחנו יודעים שגם את ההדר אתם בונים באלמנטור!
לכן הכנו רכיבי חנות מותאמים להדר האתר, אותם תוכלו לשלב בקלות בחלקי ההדר השונים.
ווידג'ט כניסת לקוחות - ההופך לאיזור אישי , עגלה מקוצרת ושדה חיפוש למוצרי החנות.

= מה כולל התוסף בקיצור =
* יצירת תבנית אלמנטור (טמפלייט) לדף המוצר
* התניות - אפשרות לשמירת תבניות שונות לדפי מוצר לפי הגדרת קטגוריה או שם המוצר.
* שילוב רכיבי מוצר עם כל רכיבי האלמנטור הקיימים , ללא הגבלה.
* רכיבים גלובליים נוספים של החנות לשילוב בחלקי האתר השונים
* רכיבים עצמאיים לשילוב בהדר


== Installation ==
1. Install Elementor & Elementor Pro
2. Install [Wizshop Plugin](https://wizshop.co.il/plugin/)
3. Install the plugin through the WordPress plugins screen directly, or upload the plugin files to the `/wp-content/plugins/wizshop` directory.
4. Activate the plugin through the 'Plugins' screen in WordPress
